Influxdb

InfluxDB用 Go 语言编写的一个开源分布式时序、事件和指标数据库，InfluxDB 提供类 SQL 语法，很容易上手，但是又和SQL有很大差别。

Infiuxdb同时也提供了 python client 方便开发，Github地址：influxdb-python。

正好最近项目中用到了Influxdb，梳理一些从0开始的过程。 Influxdb 仅仅单机版本免费，集群使用并不免费，注意。
